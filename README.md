# Your_Tempo 🎧

スマホ画面を 5 回タップすると、その平均間隔から「BPM（テンポ）」を算出。  
算出された BPM に近い楽曲を Spotify から検索し、オリジナルのプレイリストを作成する Web アプリです 🎵

---

## 🎧 利用に必要なもの

- **Spotify アカウント（無料プランで OK）**
  - 楽曲検索・プレイリスト作成に Spotify API を使用しています。
  - 初回アクセス時に、**トップページの「Spotify にログインする」ボタンから
    簡単にログイン**できます。

---

## 🔧 使用技術

- Python / Flask
- HTML / CSS / JavaScript
- Spotify Web API
